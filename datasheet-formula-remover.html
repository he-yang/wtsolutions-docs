Datasheet formula remover

A Excel add-in. Remove formula of cells from Excel, text and font are kept

1.Introduction

We use Excel to prepare datasheets, and sometimes formulas used in the sheets are supposed to be removed before sending the file to someone else (for instance a client).

This add-in can help you remove formula of selected cells, and at the same time maintaining font/color/highlight of the cells.

2.Requirements

Excel 2016 or greater on Windows system

Excel for Mac has not yet been tested

Excel Online

3.Usage

1. Select cells for formula removal

2. click on go button.

Note. It is strongly recommended to have your file backed up before processing.

4.Privacy

This Add-In will not collect and/or transmit user's document content. Baidu Tongji (similar to Google analytics) scripts is added for website analytics.

5.Disclaimer
